# Cybrilla Tester

Just a test app built at Cybrilla.
This program mimics a payment transaction.

## How to run

- Clone the repositiory.

~~~
git clone git@github.com:swaroopsm/cybrilla-test.git
~~~

- Install Dependencies

~~~
bundle install
~~~

- Start the server

~~~
rackup
~~~

- Run the application

~~~
$ ./app
~~~
